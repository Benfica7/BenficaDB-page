Tão simples quanto uma função

O BenficaDB tem seu uso baseado no uso de funções simplificadas para você.
from bdb import Database
db = Database("users.json")
db.insert_one(
    {"name": "Benfica", "age": 15}
)
db.save()

De Benfica para o mundo

O objetivo do BenficaDB é ajudar desenvolvedores sem tempo da melhor forma possivel. É simples mas, as vezes é só disso que precisamos.
  • Não requer conhecimento avançado de SQL ou outra linguagem de consulta de banco de dados.
  • Não requer a instalação de um banco de dados externo.
  • Pode ser facilmente incorporado a um projeto como uma biblioteca.
  • Pode ser usado para armazenar dados em varios formatos como JSON, que é amplamente usado e facilmente legível por humanos.
  • Não é adequado para lidar com grandes conjuntos de dados devido ao armazenamento na memória.
  • Pode ser menos eficiente em termos de desempenho em comparação com o banco de dados especializado.
